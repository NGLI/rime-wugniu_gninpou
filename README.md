# 寧波吳語注音輸入方案吳語學堂版

配方： ℞ **`wugniu_gninpou`**

基於[Rime](https://rime.im/)的寧波話吳語拼音輸入方案。

含寧波城區及鄞縣鍾公廟（鄞縣中區，今鄞州區鍾公廟街道）兩種口音的輸入方案。

拼音方案爲[吳語學堂](https://www.wugniu.com/)式拼音。

使用以下項目：

- [寧波方言資料整理](https://github.com/shinzoqchiuq/gninpou-dialect)
