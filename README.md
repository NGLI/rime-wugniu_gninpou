# 寧波吳語注音輸入方案吳語學堂版



基於[Rime中州韻輸入法引擎](https://rime.im/)的寧波話吳語拼音輸入法。

提供寧波城區及鄞縣鍾公廟（鄞縣中區，今鄞州區鍾公廟街道）兩種口音的輸入法。
